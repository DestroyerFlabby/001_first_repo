# Account Setup

Create separate business accounts for the media page and client services.

## TikTok

- Choose a clear niche name.
- Use a simple logo or wordmark.
- Write a direct bio.
- Add contact method when available.
- Post consistently before optimizing too much.

## Instagram

- Use the same handle if possible.
- Add a clear profile image.
- Use a bio that explains the page in one line.
- Pin best posts once you have winners.
- Add service CTA if using the page for leads.

## YouTube

- Create a dedicated channel.
- Use a clear banner and profile image.
- Organize Shorts around one niche.
- Avoid low-effort reposting.
- Add descriptions that explain the transformed value.

## Naming Ideas

- Build Signal.
- Founder Frames.
- Sharp Clips.
- Context Daily.
- Signal Shorts.

## Bio Templates

Media page:

`Business lessons from the best long-form conversations. Useful ideas, compressed.`

Service page:

`I turn long-form content into short clips for creators and founders. DM "clips" for samples.`

## Branding Checklist

- Handle chosen.
- Profile photo.
- Bio.
- Link destination.
- Highlight color.
- Subtitle style.
- First 10 post ideas.
- Tracking sheet ready.
