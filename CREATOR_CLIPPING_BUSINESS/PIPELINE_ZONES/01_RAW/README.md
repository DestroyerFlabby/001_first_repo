# 01 Raw

Pipeline step: acquire data.

This zone stores source material before transformation or editing.

Use it for:

- Source URLs.
- Downloaded videos.
- Creator-provided footage.
- Initial rights notes.

Do not post directly from this zone.

## Subfolders

`SOURCE_LINKS`

Source URL queues and notes.

`DOWNLOADED_MEDIA`

Raw downloaded or collected media. Large media files are ignored by Git.
