# 03 Staging

Pipeline step: editing.

This zone is for drafts that are being edited or reviewed.

Use it for:

- Draft clips.
- Draft captions.
- Hook text.
- Subtitle review.
- B-roll plans.
- Client review notes.

Nothing in this zone is considered final.

## Subfolders

`DRAFT_CLIPS`

Work-in-progress clip exports. Large media files are ignored by Git.

`CAPTIONS_AND_HOOKS`

Draft hooks, captions, descriptions, hashtags, and CTA options.
