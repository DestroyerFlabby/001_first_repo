# 04 Curated

Pipeline step: posting.

This zone is for approved assets that are ready to post or already posted.

Use it for:

- Final captions.
- Ready-to-post metadata.
- Approved clip notes.
- Posted records.
- Performance summaries.

Only content with a clear `rights_status` should reach this zone.

## Subfolders

`READY_TO_POST`

Approved assets and final posting notes.

`POSTED_AND_TRACKED`

Posted content records and performance summaries.
