# Editing Checklist

## Before Editing

- Confirm the clip has one clear idea.
- Choose the hook.
- Decide the platform format.
- Confirm target length.

## Edit

- Cut dead air.
- Keep pacing tight.
- Add clean subtitles.
- Highlight important words.
- Add hook text in the first frame.
- Add B-roll only if it improves understanding.
- Keep audio clear.
- Add brand style if needed.

## Final Review

- Does the first second create interest?
- Can the clip be understood without sound?
- Is the takeaway clear?
- Is there any copyright or repost risk?
- Does the clip add transformation?
- Is the export size correct for the platform?

## Export

- Format: 9:16 vertical.
- Resolution: 1080x1920.
- Captions readable on mobile.
- File name includes client, topic, and date.
