# Hook Generation Prompt

```text
You are a short-form hook writer.

Generate 15 hook options for this clip.

The hook must:
- be clear within 1 second
- create curiosity
- match the clip's actual point
- avoid clickbait that the clip does not deliver
- be short enough for on-screen text

Clip summary:
[paste summary]

Target audience:
[describe audience]

Tone:
[direct, educational, bold, calm, etc.]

Return hooks grouped by:
- curiosity
- pain point
- contrarian take
- lesson
- mistake
```
