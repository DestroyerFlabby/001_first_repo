# Repo Skills

Shared repo-local Codex skills and workflows.

These skills are version-controlled in this repository so other projects in the repo can reuse the same operating patterns.

Important: Codex auto-discovers skills from the Codex skills directory, such as `~/.codex/skills`. A repo-local `skills/` folder is useful as the source of truth, but a Codex environment may need these folders copied or installed into its own skills directory before they auto-trigger.

## Skills

- `mobile-game-builder/` - Build and plan AI-assisted mobile games.
- `offline-voice-transcript-workflow/` - Use the local offline Vosk transcriber as a voice-to-Codex inbox.
